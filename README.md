# To-do App

A simple and efficient to-do application built with React.

## Features
- Add new tasks
- Mark tasks as complete
- Delete tasks
- Filter tasks (All/Active/Completed)

## Setup and Installation

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn

### Installation
```bash
git clone https://github.com/veerakumari810/To-do-app.git
cd To-do-app
npm install
npm run dev
